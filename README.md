# Airhorn-Bot
### Commands:

`-airhorn` ~ plays an airhorn into the vc

more commands coming soon lol

### Setup:
1) Clone the code.
2) Create a new discord application in the (developer portal)[https://discord.com/developers/applications] and create a bot.
3) Copy the discord token given for the bot: ![image](https://user-images.githubusercontent.com/72450527/111185348-408d9880-8588-11eb-94ca-8f35220b7864.png)
4) (Install node)[https://nodejs.org/en/].
